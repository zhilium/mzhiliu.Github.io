M1 Mac 使用 Github Pages + Hexo 搭建个人博客

Posted on 2024-11-06 In CS , Others

第一次接触静态网页和 Hexo 框架，配置的过程中踩了几个坑，在此做一个简单记录。主要是为了自己复用，能帮助遇到同样问题的朋友就更好了。

我的环境：Apple M1/M2 Mac, macOS 15.1 (24B83), kernel version darwin 24.1.0.

安装 Hexo

Hexo 是一个基于 Node.js 编写的静态博客框架，可以把 markdown 格式文档解析为 html 格式的静态网页。Hexo 简洁易用，拥有活跃的社区，有很多优秀的第三方主题，使得创作者能够专注于内容产出，无需再花费精力设计复杂的界面。

推送至 Github Pages

GitHub Pages 是一个由 GitHub 提供的静态网站托管服务，允许用户直接从 GitHub 仓库中托管网站。简单来说，你可以直接把构建好的静态网页作为一个 repo 上传，然后 Github Pages 会直接解析这个 repo, 部署为网站。

创建要解析的 repo

  1. 创建一个新 repo, 命名为 <github_user_name>.github.io. 必须是这个格式，否则后续可能会解析失败。
  2. 进入 repo/Settings/Pages/Build and deployment,
    • source 选择 deploy from a branch.
    • branch 选择 /main/.
